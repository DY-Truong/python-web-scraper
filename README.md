# Python Web Scraper

A Python tool that gathers website information including metadata, DNS records, subdomains, port scans, and internal links.

## Features
- Extracts page titles and meta tags
- Performs DNS lookup
- Finds subdomains
- Runs port scans using Nmap
- Builds a simple site map

## Technologies
- Python
- Requests
- BeautifulSoup
- dnspython
- python-nmap

## Usage

Install dependencies:

pip install requests beautifulsoup4 dnspython python-nmap

Run the script:

python web_scraper.py
